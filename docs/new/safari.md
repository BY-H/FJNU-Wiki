## 校园地图
### 旗山校区（含河西校区）
注意：本份平面图修订于2020年，2025年相较于此已发生较大变化。主要新增楼栋如下：

| 名称 | 地点 | 
| --- | --- | 
| 化学与材料学院(理工楼15号楼) | 东区田径场入口正对面 | 
| 地理科学学院(理工楼16号楼) | 15号楼南边（最高的科研楼） | 
| 物理与能源学院、网络安全学院、数学与统计学院(理工楼17-18号楼) | 理工楼群旁边草坪上的小校门正对面 | 
| 学生宿舍 李苑9-11号楼 | 千叶园/文化街美食餐厅旁边 |
